## Running a C++ program
1. compiling
    ```
    g++ <file name>.cpp
    ```
    OR
    ```
    g++ <file name>.cpp -o <output file>
    ```

1. executing
    ```
    ./a.out
    ```
    OR
    ```
    ./<output file>
    ```
