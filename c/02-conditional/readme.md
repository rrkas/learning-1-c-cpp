# Conditional

## `true` and `false`
In C,
- `true` = any number other than 0
- `false` = 0

## Types of conditional statements
1. `if`
1. `if - else`
1. `if - else if - else`
1. `if - else if - else if - ... - else` (if-else ladder)
1. nested `if` statement
1. `switch` statement
1. ternary/ conditional operator (`?:`)
